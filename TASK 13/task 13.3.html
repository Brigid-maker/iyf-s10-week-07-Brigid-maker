Storage Comparison

sessionStorage

Used in this form. Data lives only for the current browser tab session.

  • Cleared when tab closes
  • Perfect for form drafts
  • Separate per tab
  • ~5MB limit

localStorage

Persists until the user or code explicitly clears it.

  • Survives tab & browser close
  • Perfect for preferences, todos
  • Shared across tabs (same origin)
  • ~5–10MB limit
